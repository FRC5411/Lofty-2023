# Lofty-2023
The in-season code for Lofty (Working name), our robot for the FRC Charged Up 2023 Season. 


## Functions
Functional Swerve drive code

## Controls
### Driver (USB 0)
#### Left Stick (X and Y axes)
Chassis Strafe
#### Right Stick (X axis)
Chassis rotate
#### A button
toggle robot/field oriented
#### B button
reset yaw to 0
